## 🎯 Deployment Steps

1. Get Gmail app password (5 min)
2. Upload to Netlify (5 min)
3. Add environment variables (2 min)
4. Test and launch! (3 min)

**See QUICK-START.md for details**

## 📧 Email Configuration

All emails are sent to: **seniorcareiq@gmail.com**

You'll receive:
- Assessment results from users
- Tour requests for facilities
- Contact form submissions

Users receive:
- Confirmation emails
- Copy of their assessment
- Tour/contact confirmations

## 🌐 After Deployment

Your site will be live at: `https://[your-name].netlify.app`

You can:
- Rename it to `seniorcareiq.netlify.app` (free)
- Add custom domain like `seniorcareiq.com` (~$12/year)

## 📱 Mobile App

The HTML works perfectly on all devices - no separate mobile app needed!

To convert to native iOS/Android app later:
- Use Capacitor (https://capacitorjs.com)
- Costs: Apple App Store ($99/year) + Google Play ($25 one-time)
